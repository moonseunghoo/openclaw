# World Market Briefing Skill

## 🌐 Overview
The `world-market-briefing` skill is a specialized analytical tool designed for the **Stock AI Manager**. It transforms raw, real-time global financial data into a professional, highly scannable, and actionable market briefing.

## ✨ Key Features
- **Automated Data Synthesis**: Combines web search results and direct web fetching for high-accuracy information.
- **Regional Coverage**: Detailed analysis across the Americas, Europe, Asia-Pacific, and South Korea.
- **Macro-to-Micro Linkage**: Connects major economic indicators (CPI, PPI, Interest Rates) and geopolitical events to specific market movements.
- **Mobile-Optimized Formatting**: Uses structured Markdown (bullets, bolding, emojis) instead of heavy tables, ensuring perfect readability on Telegram and mobile devices.

## 🚀 How to Use
Simply request a briefing in a natural language command:
- `"world-market-briefing 스킬 실행해줘"`
- `"세계증시 브리핑 해줘"`
- `"Give me a global market briefing"`

## 📝 Output Structure
1. **📈 Market Overview**: A punchy summary of the global sentiment.
2. **🌍 Regional Performance**: Scannable list of major indices with direction emojis.
3. **🔍 Key Drivers & Macro Data**: Contextual explanation of *why* markets are moving.
4. **🇰🇷 Impact on Korean Market**: Specific implications for the KOSPI/KOSDAQ.
5. **💡 Analyst Insight**: A final, actionable professional conclusion.

## 🛠️ Technical Details
- **Dependencies**: `web_search`, `web_fetch`
- **Persona**: Stock AI Manager (Sharp, Competent, Data-driven)
- **Last Updated**: 2026-05-14
